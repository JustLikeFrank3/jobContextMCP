# JobContextMCP Marketing Plan

- Tech-savvy job seekers
- Coding bootcamp graduates
- Career coaches or small teams
- Developers preparing for interviews

## Messaging
- "Zero-config AI job-hunt assistant"
- "Conversational workspace setup in minutes"
- Multi-language prep for coding interviews
- Persistent job pipeline + resume memory

## Channels
- GitHub repo / README
- LinkedIn posts
- Twitter / dev communities (Reddit, Hacker News)
- Video demos on YouTube or Loom

## Assets
- Horizontal banner for README
- Logo for social media / icons
- Demo GIFs for setup walkthroughs

## Launch Sequence
1. Publish v0.5 CLI-ready workspace
2. Post demo video + GIF
3. Announce Copilot conversational setup
4. Open early access for hosted workspace or zip package
5. Launch template packs
