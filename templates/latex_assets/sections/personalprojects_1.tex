
\subtext {U.S. Drought Classification Forecast}
\begin{zitemize}
 \item Developed a predictive model to classify US County drought levels by leveraging a 20-year dataset from NASA's POWER Project, encompassing 21 features and over 23.8 million instances, utilizing Python libraries such as Scikit-learn, Numpy, and Matplotlib.
\item Assessed the dataset using the Long Short-Term Memory (LSTM) model in conjunction with TensorFlow/Keras. Additionally, employed Scikit-learn algorithms including Random Forest, Logistic Regression, and SVM to gauge performance based on accuracy metrics.
\item Selected the LSTM model, which demonstrated superior performance with an accuracy score of 85\% on balanced datasets, utilizing categorical cross-entropy and the Adam variant of stochastic gradient descent.
\item Enhanced the robustness of the model by addressing data imbalance through techniques such as Under-Sampling, Over-sampling, and SMOTE. Employed Cross-validation, Standardized Scaling, and Grid Search for optimal hyper-parameter tuning, resulting in an improved LSTM model accuracy of 89\%.
\end{zitemize}

\subtext {Credit Card Fraud Analytics}
\begin{zitemize}
 \item Developed sophisticated algorithms to identify fraudulent credit card applications from a dataset comprising over 280,000 records with 31 features and a 3\% detection rate. Utilized Python and Scikit-learn to implement models including KNN, Gradient Boosted Trees, and Random Forest.
\item Achieved a commendable accuracy of 72\% by deploying a Multi-Layer Perceptron model, leveraging the capabilities of TensorFlow and Keras.
\item Enhanced model performance by incorporating dimensionality reduction techniques such as Wrapper methods (Step-wise and Forward selection) and Principal Component Analysis (PCA).
\item Articulated the model's results and its implications in a comprehensive business report, visualized using PowerBI.
\end{zitemize}

\subtext{Used Car Sales Analysis}
\begin{zitemize}
\item Conducted advanced data preprocessing on a dataset of over 70,000 used car listings from eBay Germany, utilizing R, to derive insights into the pricing dynamics of the German used car market.
\item Implemented a linear regression model to forecast car prices, considering influential attributes such as color, year of manufacture, mileage, and distance driven.
\item Enhanced model accuracy by employing forward step-wise regression for feature selection, achieving a commendable accuracy rate of 88%.
\end{zitemize}
